## 👨‍💻 Sobre mí

Ingeniero de Sistemas enfocado en construir soluciones escalables que conectan la excelencia técnica con el valor real de negocio. Mi objetivo es diseñar arquitecturas robustas hoy y liderar los equipos que las construirán mañana.

- ⚙️ **Arquitectura & Backend:** Diseño de sistemas distribuidos, APIs RESTful y microservicios (Java, Spring Boot).
- ☁️ **DevOps & Orquestación:** Contenedorización y despliegue automatizado (Docker, Kubernetes).
- 📊 **Data Engineering:** Creación de pipelines, validación de datos y automatización de procesos críticos.
- 🎯 **Enfoque actual:** Posicionamiento SOUL y transición fluida entre el desarrollo técnico profundo y la gestión de proyectos de ingeniería.
